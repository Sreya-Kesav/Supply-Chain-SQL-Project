# Product Analysis

## Objective

Analyze product performance to find the products that generate the highest revenue and profit.

## Business Questions

- Which products generate the highest revenue?
- Which products generate the highest profit?
- Which products generate the lowest profit?
- What is the profit margin of each product?

## SQL Concepts Used

- JOIN
- GROUP BY
- SUM()
- COALESCE()
- ORDER BY
- Views

## Files Included

- 06_Product_Analysis.sql
- 06A_Product_Analysis_Using_Views.sql

## Business Impact

This analysis helps identify the best selling products and the products that bring low profit. These insights can help with pricing, inventory, and product decisions.
