# Time Analysis

Analyze business performance over time.

## Business Questions

- How does revenue change every month?
- Average Order Value
- Which months recieve the most orders?

## SQL Concepts Used

- JOIN
- GROUP BY
- Aggregate Functions (SUM(), COUNT())
- Date Functions (YEAR(), MONTH(), DATENAME())
- COUNT(DISTINCT)
- ORDER BY
- TOP

## Files Included

- 11_Time_Analysis.sql

## Business Impact

This analysis helps understand business growth trends and identify periods of high and low performance.
