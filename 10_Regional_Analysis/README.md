# Regional Analysis

## Objective

Compare business performance across different markets, countries, and regions.

## Business Questions

- Which market generates the highest revenue?
- Which country generates the highest revenue?
- Which region generates the highest profit?
- Which countries have the highest late delivery rate?

## SQL Concepts Used

- JOIN
- GROUP BY
- Aggregate Functions
- CASE WHEN

## Files Included

- 10_Regional_Analysis.sql

## Business Impact

This analysis helps compare business performance across different locations and identify regions that need attention.
