# Data Profiling

## Objective

Examine the quality, structure, and consistency of the raw dataset.

## Business Questions

- How many records exist in the dataset?
- Are there any missing values in critical columns?
- How many unique business entities exist?
- Does each business ID consistently map to a single descriptive attribute?

## SQL Concepts Used

- COUNT()
- COUNT(DISTINCT)
- GROUP BY
- HAVING
- CASE WHEN
- Aggregate Functions

## Files Included

- `02_Data_Profiling.sql`

## Key Takeaways

Validated the dataset by checking completeness, uniqueness, and consistency to ensure reliable normalization and analysis.
