# Category Analysis

## Objective

Analyze product categories to understand which categories perform the best.

## Business Questions

- Which categories generate the highest revenue?
- Which categories generate the highest profit?
- What is the profit margin for each category?
- How much does each category contribute to total revenue?

## SQL Concepts Used

- JOIN
- GROUP BY
- SUM()
- Subquery
- COALESCE()
- ORDER BY

## Files Included

- 07_Category_Analysis.sql

## Business Impact

This analysis helps understand which product categories contribute the most to the business and which categories may need improvement.
