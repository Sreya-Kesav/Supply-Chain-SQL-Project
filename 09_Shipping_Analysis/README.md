# Shipping Analysis

## Objective

Analyze shipping performance and delivery delays.

## Business Questions

- What is the average shipping time?
- How many orders were delivered late?
- Which shipping mode is used the most?
- What is the average delivery delay?

## SQL Concepts Used

- CASE WHEN
- AVG()
- COUNT()
- GROUP BY

## Files Included

- 09_Shipping_Analysis.sql

## Business Impact

This analysis helps understand delivery performance and identify possible issues in the shipping process.
